TYPES OF PSEUDO-CLASS

Pseudo-class is used to style an element that are in specific condition,it is denoted by":"

  • hover
  • active
  • focus
  • link
  • visited
Hover
As discussed earlier, this pseudo-class is used to apply styles to an element when it's being hovered over by the cursor.
Active
This pseudo-class applies styles to an element while it's being activated by the user, typically when it's being clicked or tapped
Focus
Styles applied to an element that currently has keyboard focus. This is commonly used in form elements like input fields and buttons.

Link
This pseudo-class selects links that have been unvisited by the user
Link

Visited
This pseudo-class selects links that have been visited by the user. It's often used to style visited links differently from unvisited ones.
