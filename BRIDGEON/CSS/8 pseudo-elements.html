TYPES OF PSEUDO-ELEMENTS

Pseudo-element is used to style an specific part of the selected elements"::"

  • first-line
  • first-letter
  • before
  • after
  • selecton
first-line
This pseudo-element targets the first line of text within the selected element. It's commonly used to style the first line of a paragraph differently from the rest of the text.

first-letter
Targets the first letter of the text within the selected element. It's often used to create drop caps or apply special styling to the first letter of a paragraph.
before
This pseudo-element inserts content before the content of the selected element. It's often used to add decorative elements or additional information to an element
after
Similar to ::before, this pseudo-element inserts content after the content of the selected element.
selection
Styles the portion of text that has been selected by the user.
